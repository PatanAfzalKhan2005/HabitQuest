Create a full-stack gamified learning web application inspired by GeeksforGeeks, LeetCode, and habit tracking platforms.

APPLICATION FLOW:

1. LANDING PAGE:

* When the user opens the website, display a modern, attractive landing page.
* Show a motivational heading like:
  "Welcome to Gain Discipline"
* Include:
  → Login button
  → Register button
* Clean UI, centered layout, smooth animations.

2. AUTHENTICATION:

* User can Register and Login.
* After login:
  → Show user's name
  → Display a motivational quote (e.g., "Welcome to Gain Discipline, stay consistent 🔥")

3. NAVIGATION (TOP RIGHT MENU):

* Add a 3-line hamburger menu with options:
  → Home
  → Aptitude
  → Coding
  → Puzzle Games
  → Profile
  → Notifications

TOPICS:

FLOW:

* When user clicks "Aptitude":
  → Show all topics in card layout
* When user clicks a topic:
  → Show 3 levels:
  🟢 Simple
  🟡 Hard
  🔴 Difficult

QUESTION STRUCTURE:

* Each level must contain EXACTLY 12 MCQs
* Total per topic = 36 questions

Each MCQ must include:

* Question
* 4 options (A, B, C, D)
* Correct answer
* Simple explanation (beginner-friendly)

QUIZ INTERACTION:

* Show one question at a time OR all 12 (configurable)
* On correct answer:
  → Highlight GREEN
  → Show confetti 🎉 animation
* On wrong answer:
  → Highlight RED
* Disable re-selection
* Show progress (e.g., Question 5/12)
* Auto-submit when time ends OR user submits

RESULT SCREEN:

* Show score (e.g., 9/12)
* Show performance message:
  → "Great job!" / "Keep practicing!"
* Show ribbons/confetti animation
* Show earned points

APTITUDE STREAK SYSTEM:

* If user solves ≥1 quiz per day:
  → Streak increases
* If missed:
  → Streak resets

Display:

* Current streak 🔥

* Calendar heatmap (monthly)

APTITUDE POINTS SYSTEM:

* +10 per correct answer

* +100 streak bonus

APTITUDE DASHBOARD (RIGHT PANEL):

  → Problems Attempted

REDEEM SYSTEM:

  → Premium questions
